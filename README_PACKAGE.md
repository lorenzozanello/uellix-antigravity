# Uellix — Paquete documental v1.0 para Antigravity

Este paquete contiene la documentación base para iniciar el desarrollo de Uellix en un repositorio limpio usando:

- Antigravity como entorno principal de desarrollo.
- GitHub como fuente de verdad.
- Vercel como plataforma de previews y despliegue.
- Next.js, TypeScript, Tailwind CSS, shadcn/ui.
- Supabase Postgres, Supabase Auth, Supabase Storage y Row Level Security.
- Drizzle como ORM.
- Gemini API como proveedor de IA para Stella.

## Promesa central

**Uellix convierte el impacto social en evidencia defendible.**

## Definición estratégica

Uellix es una plataforma SaaS B2B de inteligencia de impacto social que ayuda a organizaciones, financiadores y equipos de sostenibilidad a convertir narrativas, evidencias, indicadores, proxies financieros y cálculos de impacto en un rastro SROI trazable, auditable y audit-ready.

## Regla de posicionamiento

Uellix no certifica automáticamente el impacto social. Uellix prepara, estructura, fortalece y documenta el análisis para que pueda ser revisado, defendido, comunicado y auditado con rigor.

## Cómo usar este paquete

1. Crear un repositorio nuevo y limpio en GitHub.
2. Crear el proyecto local o abrir el repositorio en Antigravity.
3. Copiar `ANTIGRAVITY.md` en la raíz del repo.
4. Copiar la carpeta `/docs` completa en la raíz del repo.
5. Abrir Antigravity y ejecutar el primer prompt de `/docs/18_PROMPTS_FOR_ANTIGRAVITY.md`.
6. No permitir instalación de dependencias ni modificación de código hasta que Antigravity entregue diagnóstico y plan de Sprint 0.
